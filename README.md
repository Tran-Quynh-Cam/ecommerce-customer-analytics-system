# E-commerce Customer Analytics System  
Customer Behavior & Satisfaction Analysis using Data Warehouse & Decision Support System  

## 📖 Overview
This project analyzes customer behavior and satisfaction in an e-commerce environment using data warehousing and decision support techniques.

The system integrates customer reviews, transaction data, and behavioral patterns to extract business insights and support managerial decision-making.

---

## 🎯 Objectives
- Analyze customer purchasing behavior
- Evaluate customer satisfaction levels
- Perform sentiment analysis on reviews
- Detect potential spam reviews
- Provide actionable business insights

---

## 🏗 System Architecture
1. Data Collection Layer
2. Data Warehouse Design (Star Schema)
3. ETL Process
4. Analytical Layer
5. Decision Support Dashboard

---

## 📊 Analytical Components
- Customer Segmentation
- Purchase Behavior Analysis
- Customer Satisfaction Metrics
- Sentiment Analysis
- Spam Review Detection
- Data Visualization
